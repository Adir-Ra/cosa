Requests

In order to have your project be considered for membership in COSA, you must complete an application with all required information and return it to the email in the application. This will then be evaluated for inclusion into the COSA directory.

The COSA board evaluates each project against the criteria below, then votes on whether or not the project should be included in the directory. The criteria are based on the Open Source Initiative 10 criteria for defining open source (https://opensource.org/osd).
